- **H1: Premium Design System**: Create a unified design system in `index.css` with a curated color palette, modern typography (Outfit/Inter), and translucent "Glassmorphism" elements.
- **H2: Home Feed "Wow" Factor**: Redesign the main dashboard to feel more dynamic, alive, and interactive.

## 2. Monetization & Growth

- **M3: Referral System Polish**: Finalize the referral rules logic to incentivize user growth.

## 3. Reliability & Maintenance
- **R1: Zero-Bug Philosophy**: Resolve current Gradle sync issues permanently.
- **R2: Automated Verification**: Fix Playwright E2E tests to ensure critical flows (Prayer, Hatim) never break in production.

## Constraints
- Must work across Web and Android (Capacitor).
- Must respect existing Firebase architecture.
- Performance must remain high on mobile devices.
