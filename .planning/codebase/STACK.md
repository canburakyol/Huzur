# Technology Stack - Projem

## Core
- **Frontend Framework**: React 19.2.0 (Vite 7.3.1)
- **Runtime/Bridge**: Capacitor 8.0.1
- **Language**: JavaScript (ESM) / TypeScript (Capacitor Config)
- **Styling**: Vanilla CSS

## Key Dependencies
- **State Management/Hooks**: React Context API, Custom Hooks
- **Navigation**: React Router Dom 7.13.0
- **Utilities**: date-fns 4.1.0, axios 1.13.2, adhan 4.4.3
- **UI Components**: Lucide React 0.556.0, Leaflet 1.9.4
- **Localization**: i18next 25.7.3, react-i18next 16.5.1
- **Visual Effects**: canvas-confetti 1.9.4, html2canvas 1.4.1

## Build & Tooling
- **Build Tool**: Vite
- **Linting**: ESLint 9.39.1
- **Testing**: Vitest 4.0.18, Playwright (E2E)
- **PWA**: vite-plugin-pwa

## Configuration Files
- `package.json` - Dependencies and scripts
- `vite.config.js` - Vite build settings
- `capacitor.config.ts` - Capacitor app settings
- `eslint.config.js` - Lint rules
- `playwright.config.js` - E2E test settings
