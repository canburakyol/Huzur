# Huzur - Prayer and Daily Islamic Companion

Huzur is an Islamic productivity app that combines prayer routines, Quran learning, and daily spiritual habits in one mobile-first experience.

## Core features

- Daily prayer times and reminders
- Quran reading modules and structured learning flows
- Dua and dhikr tracking
- Qibla and mosque-related utility tools
- Habit and streak systems for consistency
- Multi-language support

## Tech stack

- React 19
- Vite 7
- Capacitor 7
- Firebase
- RevenueCat
- JavaScript / TypeScript

## Local setup

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Android sync/open:

```bash
npx cap sync android
npx cap open android
```
